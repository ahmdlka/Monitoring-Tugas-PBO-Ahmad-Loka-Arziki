Tugas 1: Pengenalan Class dan Object

12 September 2025
Waktu pengerjaan: 3 hari
Class Object Java Dasar Pemrograman

Deskripsi Tugas

Pada tugas pertama mata kuliah Pemrograman Berbasis Objek, saya mempelajari konsep dasar class dan object. Tugas ini mengharuskan saya untuk membuat program sederhana dalam bahasa Java yang mendemonstrasikan pembuatan class, pembuatan object, serta penggunaan atribut dan method.

Implementasi Program

Saya membuat program sederhana untuk mendefinisikan class Mahasiswa dengan atribut nama, NRP, dan jurusan. Class ini juga memiliki method untuk menampilkan data mahasiswa. 

public class Mahasiswa {
    // Atribut
    private String nama;
    private String nrp;
    private String jurusan;
    
    // Constructor
    public Mahasiswa(String nama, String nrp, String jurusan) {
        this.nama = nama;
        this.nrp = nrp;
        this.jurusan = jurusan;
    }
    
    // Method untuk menampilkan data
    public void tampilkanData() {
        System.out.println("Nama: " + this.nama);
        System.out.println("NRP: " + this.nrp);
        System.out.println("Jurusan: " + this.jurusan);
    }
    
    // Main method untuk testing
    public static void main(String[] args) {
        Mahasiswa mhs1 = new Mahasiswa("Ahmad Loka", "5025211234", "Informatika");
        mhs1.tampilkanData();
    }
}

Penjelasan Konsep

Dalam pemrograman berorientasi objek, class adalah blueprint atau cetakan untuk membuat object. Class mendefinisikan atribut (data) dan method (perilaku) yang dimiliki oleh object. Object adalah instance dari class yang memiliki state (nilai atribut) dan behavior (method) tertentu.

Pada contoh program di atas, Mahasiswa adalah class yang memiliki tiga atribut (nama, nrp, jurusan) dan satu method (tampilkanData). Object mhs1 adalah instance dari class Mahasiswa dengan nilai atribut tertentu.

Kesimpulan

Tugas ini memberikan pemahaman dasar tentang class dan object dalam pemrograman berorientasi objek. Saya belajar bagaimana mendefinisikan class dengan atribut dan method, membuat object dari class, serta mengakses atribut dan method dari object. Konsep ini merupakan fondasi penting untuk mempelajari konsep-konsep OOP yang lebih advanced seperti inheritance, polymorphism, dan encapsulation.
