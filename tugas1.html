Tugas 1: Typing speed test

26 Agustus 2025
Waktu pengerjaan: 2 hari
Typing Hard Skill

Deskripsi Tugas

Pada tugas pertama ini, saya melakukan typing speed test untuk mengukur kecepatan serta akurasi dalam mengetik menggunakan keyboard komputer. Tes ini penting untuk menunjang aktivitas sehari-hari sebagai mahasiswa Informatika, terutama ketika menulis kode program maupun laporan.

Untuk melakukan tes, saya menggunakan situs 10FastFingers.com yang memang dirancang khusus untuk menguji kemampuan mengetik. Website ini menampilkan serangkaian kata acak yang harus diketik dalam batas waktu tertentu, kemudian menghasilkan skor berupa kecepatan mengetik (dalam WPM – words per minute) serta tingkat akurasi. Dari hasil ini, saya dapat mengetahui sejauh mana kemampuan mengetik saya dan apa yang masih bisa ditingkatkan.

Kesimpulan

Melalui tugas ini saya menyadari bahwa kemampuan mengetik yang cepat dan akurat sangat bermanfaat untuk mendukung produktivitas, khususnya dalam dunia pemrograman. Dengan latihan dan evaluasi hasil tes di 10FastFingers.com, saya dapat mengetahui performa mengetik saya saat ini dan menjadikannya dasar untuk meningkatkan kecepatan maupun akurasi. Kebiasaan ini akan membantu saya bekerja lebih efisien saat menulis kode maupun membuat dokumen.
