Tugas 6: Tech Support System Menggunakan Library Java

13 October 2025
Waktu pengerjaan: 1 hari
Java HashMap ArrayList PBO

Deskripsi Tugas

Pada era digital saat ini, sistem dukungan teknis (tech support system) menjadi komponen penting dalam memberikan layanan pelanggan yang efisien. Banyaknya perangkat lunak dan perangkat keras yang digunakan oleh individu maupun organisasi menuntut adanya sistem bantuan yang responsif dan terstruktur.

Pada tugas ini saya membuat sebuah sistem bantuan teknis sederhana menggunakan Java dengan memanfaatkan library seperti Scanner, ArrayList, HashMap, dan Random. Sistem ini dapat memberikan respons otomatis berdasarkan kata kunci tertentu yang diberikan pengguna.

Implementasi Program

Berikut adalah implementasi lengkap dari Tech Support System:

Class InputReader

import java.util.Scanner;

public class InputReader {
    private Scanner reader;

    public InputReader() {
        reader = new Scanner(System.in);
    }

    public String getInput() {
        System.out.print("> ");
        String inputLine = reader.nextLine().trim().toLowerCase();
        
        return inputLine;
    }
}

Kelas InputReader bertugas membaca input dari pengguna. Metode getInput() mengembalikan teks yang sudah dipangkas spasi dan diubah menjadi huruf kecil agar mudah diproses.

Class Responder

import java.util.HashMap;
import java.util.ArrayList;
import java.util.Random;

public class Responder { 
    private HashMap responseMap;
    private ArrayList defaultResponses;
    private Random randomGenerator;

    public Responder() {
        responseMap = new HashMap<>();
        defaultResponses = new ArrayList<>();
        fillResponseMap();
        fillDefaultResponses();
        randomGenerator = new Random();
    }

    public String generateResponse(String inputLine) {
        String response = responseMap.get(inputLine);
        if(response != null) return response;
        else return pickDefaultResponse();
    }

    private void fillResponseMap() {
        responseMap.put("login error", "Baik, Anda mengalami kesalahan login. Bisa sebutkan pesan error yang muncul?");

        responseMap.put("tidak bisa login", "Baik, Anda tidak dapat login. Apakah muncul pesan tertentu di layar?");
        
        responseMap.put("lupa password", "Silakan gunakan fitur reset password. Jika masih gagal, beri tahu saya.");
        
        responseMap.put("server down", "Terima kasih atas laporannya. Apakah semua layanan tidak dapat diakses atau hanya sebagian?");
        
        responseMap.put("jaringan lambat", "Baik, apakah kelambatan terjadi di seluruh perangkat atau hanya satu?");
        
        responseMap.put("internet mati", "Baik, mohon cek sambungan kabel atau WiFi. Apakah lampu indikator router normal?");
        
        responseMap.put("aplikasi error", "Baik, aplikasi mengalami error. Bisa informasikan langkah sebelum error terjadi?");
        
        responseMap.put("bug", "Untuk laporan bug, mohon sertakan langkah reproduksi dan tangkapan layar jika ada.");
        
        responseMap.put("install software", "Tentu, software apa yang ingin Anda pasang? Pastikan Anda memiliki izin instalasi.");
        
        responseMap.put("update gagal", "Baik, pembaruan gagal. Apakah ada kode error yang muncul?");
        
        responseMap.put("printer tidak terdeteksi", "Baik, apakah printer terhubung dengan benar dan driver sudah terpasang?");
        
        responseMap.put("wifi gagal connect", "Apakah perangkat lain dapat tersambung ke WiFi tersebut?");
    }

    private void fillDefaultResponses() {
        defaultResponses.add("Maaf, saya belum dapat memahami masalah tersebut. Bisa dijelaskan lebih detail?");
        defaultResponses.add("Informasi yang Anda berikan belum dapat saya proses. Mohon jelaskan dengan lebih spesifik.");
        defaultResponses.add("Untuk membantu lebih lanjut, saya membutuhkan penjelasan tambahan.");
        defaultResponses.add("Permintaan tersebut belum tersedia dalam sistem. Silakan coba dengan kata lain.");
        defaultResponses.add("Saya belum memiliki jawaban untuk itu. Coba ajukan pertanyaan yang lebih teknis.");
    }

    private String pickDefaultResponse() {
        int index = randomGenerator.nextInt(defaultResponses.size());
        return defaultResponses.get(index);
    }
}

Responder bertugas menghasilkan jawaban. Jika kata kunci cocok dengan data di HashMap, maka sistem mengembalikan jawaban tersebut. Jika tidak, ia mengambil jawaban acak dari defaultResponses.

Class SupportSystem

public class SupportSystem {
    private InputReader reader;
    private Responder responder;
    
    public SupportSystem() {
        reader = new InputReader();
        responder = new Responder();
    }

    public static void main(String[] Args) {
        SupportSystem app = new SupportSystem();
        app.run();
    }

    public void run() {
        boolean finished = false;
        printWelcome();

        while(!finished) {
            String input = reader.getInput();

            if(input.equals("selesai")) finished = true;
            else {
                String response = responder.generateResponse(input);
                System.out.println(response);
            }
        }
        printGoodbye();
    }

    public static void printWelcome() {
        System.out.println("==============================================");
        System.out.println("        Selamat Datang di Support Assistant");
        System.out.println("==============================================");
        System.out.println("Saya adalah asisten dukungan teknis Anda.");
        System.out.println("Silakan ajukan masalah atau pertanyaan yang ingin Anda selesaikan.");
        System.out.println();
    }

    private void printGoodbye() {
        System.out.println("Senang berbicara dengan anda...");
    }
}

Kelas SupportSystem adalah inti dari program. Ia menggabungkan InputReader untuk membaca input dan Responder untuk memberikan jawaban otomatis.

Penjelasan Konsep

  • HashMap → menyimpan pasangan pertanyaan–jawaban.
  • ArrayList → menyimpan jawaban default.
  • Random → memilih respons acak jika input tidak cocok.
  • Scanner → membaca input dari pengguna.

Kesimpulan

Program Tech Support System ini menunjukkan bagaimana library Java dapat digunakan untuk membuat sistem respons otomatis yang sederhana namun efektif. Dengan pemanfaatan HashMap, ArrayList, dan Random, aplikasi dapat memberikan jawaban yang konsisten, atau jawaban acak jika tidak mengenali input.
