Tugas 9: Vehicle Rental System – Inheritance

8 November 2025
Waktu pengerjaan: 1 hari
Java Inheritance OOP Vehicle System

Deskripsi Tugas

Pada proyek ini, saya mengembangkan sebuah sistem manajemen penyewaan kendaraan yang dibangun menggunakan bahasa Java. Fokus utama dari tugas ini adalah penerapan konsep pewarisan (inheritance) untuk memodelkan berbagai jenis kendaraan dengan karakteristik berbeda. Setiap kelas turunan mewarisi atribut dasar dari kelas induk, namun tetap memiliki ciri khasnya masing-masing, sehingga struktur program menjadi lebih terorganisir dan mudah dikembangkan.

Fitur Utama

Sistem ini menyediakan beberapa kemampuan inti, antara lain:

  • Menambahkan kendaraan baru ke dalam daftar.
  • Menghapus kendaraan yang tidak lagi tersedia.
  • Mencari kendaraan berdasarkan kriteria tertentu.
  • Melakukan proses penyewaan dan pengembalian kendaraan.

Setiap jenis kendaraan—mobil, motor, maupun sepeda—dibuat sebagai kelas turunan dengan atribut unik, namun tetap berbagi struktur dasar dari kelas Vehicle.

Struktur Kelas

Berikut gambaran umum relasi antar kelas dalam sistem:

  • Vehicle – kelas induk yang menyimpan atribut umum semua kendaraan.
  • Car – kelas turunan yang menambahkan atribut seperti jumlah pintu dan kursi.
  • Motorcycle – kelas turunan yang menambahkan kapasitas mesin dan status listrik.
  • Bike – kelas turunan yang menambahkan tipe sepeda dan informasi carrier.

Kode Program

Berikut adalah implementasi lengkap dari sistem penyewaan kendaraan menggunakan konsep inheritance dalam Java:

Class InputReader

import java.util.Scanner;

public class InputReader
{
    private Scanner scanner;

    public InputReader() {
        scanner = new Scanner(System.in);
    }

    public String readString(String prompt) {
        System.out.print(prompt);
        return scanner.nextLine();
    }

    public void close() {
        scanner.close();
    }
}

Penjelasan:
Kelas InputReader bertugas menangani seluruh proses pengambilan input dari pengguna. Dengan memisahkan logika input ke dalam kelas tersendiri, kode utama menjadi lebih bersih serta mudah dipelihara. Pendekatan ini memastikan bahwa interaksi pengguna tidak bercampur dengan logika pengelolaan data kendaraan.

Class Vehicle

public class Vehicle
{
    private String make;
    private String model;
    private int year;

    public Vehicle(String make, String model, int year) {
        this.make = make;
        this.model = model;
        this.year = year;
    }

    public String getMake() {
        return make;
    }

    public String getModel() {
        return model;
    }

    public int getYear() {
        return year;
    }

    public void showDetail() {
        System.out.println("Make: " + make);
        System.out.println("Model: " + model);
        System.out.println("Year: " + year);
    }
}

Penjelasan:
Vehicle adalah fondasi dari seluruh jenis kendaraan pada sistem ini. Kelas ini menyimpan atribut dasar, seperti make, model, dan year, yang dimiliki seluruh kendaraan. Dengan menetapkan atribut umum di kelas induk, kelas turunan dapat fokus menambahkan karakteristik spesifik tanpa harus menulis ulang kode.

Class Car

public class Car extends Vehicle
{
    private int numberOfDoors;
    private int numberOfSeats;
    private int numberOfWheels;
    private boolean isElectric;

    public Car(String make, String model, int year, int numberOfDoors, int numberOfSeats, int numberOfWheels, boolean isElectric) {
        super(make, model, year);
        this.numberOfDoors = numberOfDoors;
        this.numberOfSeats = numberOfSeats;
        this.numberOfWheels = numberOfWheels;
        this.isElectric = isElectric;
    }

    public int getNumberOfDoors() {
        return numberOfDoors;
    }

    public int getNumberOfSeats() {
        return numberOfSeats;
    }

    public int getNumberOfWheels() {
        return numberOfWheels;
    }

    public boolean isElectric() {
        return isElectric;
    }

    @Override
    public void showDetail() {
        super.showDetail();
        System.out.println("Number of Doors: " + numberOfDoors);
        System.out.println("Number of Seats: " + numberOfSeats);
        System.out.println("Number of Wheels: " + numberOfWheels);
        System.out.println("Is Electric: " + isElectric);
    }
}

Penjelasan:
Kelas Car memperluas kelas induk dengan atribut tambahan seperti jumlah pintu, kursi, roda, hingga status kendaraan listrik. Dengan melakukan override pada metode showDetail(), kelas ini mampu menampilkan informasi lengkap mobil sekaligus menggabungkan detail dasar dari kelas Vehicle.

Class Motorcycle

public class Motorcycle extends Vehicle
{
    private boolean isElectric;
    private int engineCapacity;

    public Motorcycle(String make, String model, int year, boolean isElectric, int engineCapacity) {
        super(make, model, year);
        this.isElectric = isElectric;
        this.engineCapacity = engineCapacity;
    }

    public boolean isElectric() {
        return isElectric;
    }

    public int getEngineCapacity() {
        return engineCapacity;
    }

    @Override
    public void showDetail() {
        super.showDetail();
        System.out.println("Is Electric: " + isElectric);
        System.out.println("Engine Capacity: " + engineCapacity + " cc");
    }
}

Penjelasan:
Motorcycle menambahkan informasi khusus untuk sepeda motor, seperti kapasitas mesin dan status kelistrikan. Pewarisan dari kelas Vehicle memungkinkan kelas ini memanfaatkan struktur dasar sekaligus menyisipkan detail khas sepeda motor.

Class Main

import java.util.ArrayList;
import java.util.HashMap;

public class Main
{
    private ArrayList vehicleList = new ArrayList<>();
    private HashMap rentor = new HashMap<>();
    private InputReader inputReader = new InputReader();

    public static void main(String[] args) {
        Main app = new Main();

        app.addCar("Toyota", "Camry", 2020, 4, 5, 4, false);
        app.addCar("Tesla", "Model S", 2021, 4, 5, 4, true);
        app.addMotorcycle("Harley-Davidson", "Street 750", 2019, false, 750);
        app.addMotorcycle("Zero Motorcycles", "SR/F", 2022, true, 110);
        app.addBike("Giant", "Escape 3", 2021, "Road Bike", true);
        app.addBike("Trek", "Marlin 7", 2020, "Mountain Bike", false);

        app.Run();
    }

    public void Run(){
        while(true){
            showGreetings();
            String input = inputReader.readString("> ");
            if(input.equals("1")){
                showList();
            }
            else if(input.equals("2")){
                showRentor();
            }
            else if(input.equals("3")){
                System.out.print("Enter the index of the vehicle to rent: ");
                int index = Integer.parseInt(inputReader.readString(""));
                System.out.print("Enter the name of the renter: ");
                String renterName = inputReader.readString("");
                addRentor(index, renterName);
            }
            else if(input.equals("4")){
                inputReader.close();
                return;
            }
        }
    }

    public void addCar(String make, String model, int year, int numberOfDoors, int numberOfSeats, int numberOfWheels, boolean isElectric) {
        Car car = new Car(make, model, year, numberOfDoors, numberOfSeats, numberOfWheels, isElectric);
        vehicleList.add(car);
    }

    public void addMotorcycle(String make, String model, int year, boolean isElectric, int engineCapacity){
        Motorcycle motorcycle = new Motorcycle(make, model, year, isElectric, engineCapacity);
        vehicleList.add(motorcycle);
    }

    public void addBike(String make, String model, int year, String type, boolean hasCarrier){
        Bike bike = new Bike(make, model, year, type, hasCarrier);
        vehicleList.add(bike);
    }

    public void addRentor(int index, String rentorName) {
        if (index >= 0 && index < vehicleList.size()) {
            Vehicle v = vehicleList.get(index);
            if (!rentor.containsKey(v)) {
                rentor.put(v, rentorName);
                System.out.println("Vehicle successfully rented to " + rentorName);
            } else {
                System.out.println("Vehicle is already rented");
            }
        } else {
            System.out.println("Vehicle index out of range");
        }
    }

    public void showList(){
        System.out.println("========= Available Vehicles =========");
        for (Vehicle v : vehicleList) {
            if (!rentor.containsKey(v)) { 
                v.showDetail();
                System.out.println("--------------------");
            }
        }
    }
    
    public void showRentor() {
        System.out.println("========= Rented Vehicles =========");
        for (Vehicle v : rentor.keySet()) {
            v.showDetail();
            System.out.println("Rented by: " + rentor.get(v));
            System.out.println("--------------------");
        }
    }

    public void showGreetings(){
        System.out.println("Welcome to vehicle rental shop");
        System.out.println("Choose an option: ");
        System.out.println("1. Show available vehicles");
        System.out.println("2. Show list of renters");
        System.out.println("3. Rent a vehicle");
        System.out.println("4. Exit");
    }
}

Penjelasan:
Kelas Main mengelola seluruh alur aplikasi, mulai dari pembuatan daftar kendaraan, penyimpanan data penyewa, hingga pemrosesan penyewaan. Kombinasi ArrayList dan HashMap digunakan untuk mengelola data kendaraan serta hubungan antara kendaraan dan penyewa secara efisien.

Kesimpulan

Melalui proyek ini, penerapan konsep inheritance benar-benar membantu dalam membangun struktur program yang rapi dan modular. Dengan menempatkan atribut umum pada kelas induk dan menambahkan karakteristik unik pada kelas turunan, sistem menjadi lebih mudah dikembangkan dan dipelihara.
